Bugfree Project Task Progress Bar
================================

This module adds:

* A progress bar on each Project Kanban card (percent + done/total)
* A PercentPie Tasks smart button with Open/Total counter (keeps the default click action)

Documentation
-------------

See ``docs/INDEX.md`` (English-first + full translations).

Post-install / upgrade checklist
--------------------------------

* Restart the Odoo service after install/upgrade.
* Clear browser cache and Odoo asset bundles (purge CDN assets if applicable).
* Tip: use ``?debug=assets`` to force asset reload.
