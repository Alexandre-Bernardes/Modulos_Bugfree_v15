# Roteiro de vídeo - Bugfree Project Task Progress Pie (Odoo 12)

## Duração alvo
2 a 3 minutos

## Estrutura
1. Problema (15s): "Sem abrir as tarefas, você não sabe se o projeto está andando."
2. Solução (30s): mostrar o smart button com percent pie e abertas/total.
3. Demo (60s): abrir 2 projetos (um 20%, outro 80%), clicar no smart button e abrir tarefas.
4. Instalação (20s): instalar, reiniciar Odoo, limpar cache.
5. Encerramento (10s): "Visibilidade imediata, UX padrão, sem JS/CSS extra."

## Texto sugerido (voice-over)
"Com o Bugfree Project Task Progress Pie, você vê o progresso das tarefas do projeto direto no smart button.
O círculo mostra o percentual, e o contador mostra abertas e total.
E ao clicar, você abre as tarefas do projeto como no Odoo padrão."
