Bugfree zTree Assets

A shared dependency module for Odoo 12 that provides the ztree_select widget for Many2one fields.

Key features

  • Tree-based selection dialog for hierarchical models (parent/children).
  • Simple XML usage: widget="ztree_select" on Many2one fields.
  • Designed to be installed once and reused by multiple modules.

Post-installation

After installing/updating, restart Odoo and clear browser/CDN caches. If needed, use ?debug=assets to validate asset loading.
