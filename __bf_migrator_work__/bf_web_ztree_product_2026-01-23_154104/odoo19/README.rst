Bugfree zTree Widget - Product
=====================

Apply zTree selection widget on Product Category fields.

Features
--------
- Injects widget=ztree_select at render-time (no XML view inheritance), to avoid XPath brittleness.
- Applies only to targeted models/fields for stability.
- Toggle via Settings.

Post-install
-----------
- Restart Odoo service.
- Upgrade module if updating.
- Clear browser/CDN cache; use ?debug=assets if needed.
